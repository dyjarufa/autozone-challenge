# React + TypeScript + Vite

## Running the Project

This project uses `pnpm` as the package manager and `Vite` as the bundler.

### Prerequisites

Ensure you have `pnpm` installed. If not, you can install it using the following command:

```bash
npm install -g pnpm
```

```bash
pnpm install
```

```bash
pnpm run dev
```
